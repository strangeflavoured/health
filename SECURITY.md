# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| main    | ✅        |

## Reporting a Vulnerability

Please do not open a public GitHub issue for security vulnerabilities.

Report vulnerabilities by opening a private vulnerability report at https://github.com/strangeflavoured/health/security/advisories/new

You can expect acknowledgement within 7 days and a resolution timeline within 14 days for critical issues.
